# PWA Icons

Du behöver skapa följande ikoner för PWA-support:

## Nödvändiga ikoner:

1. **icon-192.png** - 192x192 px
2. **icon-512.png** - 512x512 px

## Design rekommendationer:

- Använd TheHUB logotyp/branding
- Bakgrund: Solid färg eller transparent
- Format: PNG med transparens om möjligt
- Färger: Matcha varumärket (accent color #2563EB)

## Verktyg för att skapa ikoner:

- **Online:** https://realfavicongenerator.net/
- **Design:** Figma, Canva, eller Adobe Illustrator
- **Från SVG:** Använd Inkscape eller online SVG to PNG converters

## Placeholder:

Tills ikoner skapas kan du använda en enkel logotyp eller placeholder.
