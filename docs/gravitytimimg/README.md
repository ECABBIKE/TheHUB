# GravityTiming

Offline-first timing software for GravitySeries cycling competitions.
Replaces Neptron/SiTiming. Uses existing SPORTident AIR+ hardware.

**Status**: Pre-development. Spec complete. Ready to build.

Python 3.11 路 CustomTkinter 路 SQLite 路 WebSocket 路 PyInstaller

```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```
